# Тестовое задание

## Установка

#### 1) Клонировать репозиторий

#### 2) Установить зависимости
    $ npm install

##### 3) Запустить проект
    $ npm start - для windows
    $ npm run-script start_linux - для linux

#### 4) Так как svg эллементы подключены через спрайты без запуска веб сервера они отображатся не будут
